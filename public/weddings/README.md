## Tips for Best Results
- Use **DALL-E 3, Midjourney, or Stable Diffusion** for generation
- Add "digital illustration, soft watercolor style, warm lighting" to prompts
- Generate at **1024x768** or **1280x960** resolution
- After generating, place files in `public/weddings/` folder
- The carousel automatically picks up the images — no code changes needed
- If an image fails to load, the gradient placeholder shows instead
